# infosec-acep-uaf-edu-website
ACEP InfoSec Website
- infosec.acep.uaf.edu

# Installation

```
git clone https://github.com/acep-uaf/infosec-acep-uaf-edu-website.git
```

```
cd infosec-acep-uaf-edu-website
```

```
npm install
```

# Running Website

```
npm start
```

Point your browser to:
- http://127.0.0.1:8080/
